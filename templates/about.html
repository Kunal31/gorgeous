About Us

Online Salon Management system is an online website for Gorgeous beauty salon which provides beauty services for females and clients can book appointment and review the services of the order using dashboards.

Using the system users or clients can view the services provided by salon and can book appointment through appointment booking page. On checkout time user can register or existing user can use their existing credentials proceed the checkout. Using booking section, Clients can view their upcoming beauty sessions.

This system has administration section where admin can access all bookings, add/view/edit services to the booking portal. As of now, the system offers only booking options without any pre-payment.
